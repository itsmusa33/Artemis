Artemis is an educational projectile motion simulator with focus on millitary artileries.



Overview

Artemis simulates realistic artillery projectile motion with accurate physics. 
Users can select from multiple weapons and ammunition types, adjust launch 
parameters and environment variables, and test their aim in multiple 
simulation modes including stationary and moving target challenges.



Features

- Angle, gravity and wind sliders with Earth/Moon presets
- Three weapons (Mortar, Field Gun, Missile System) plus custom weapon
- Three shell types (50mm, 100mm, 150mm) each with unique blast radius
- Real-time trajectory arc accounting for wind and gravity
- Live stats — range, max height, flight time
- Practice Mode, Static Challenge and Moving Target Challenge
- Auto-Aim assist, crater visualization, hit/miss/breach HUD
- Keyboard controls — A/D for angle, M to launch, F11 for fullscreen
- Launch and impact sound effects

Running the Project

1. Clone the repository
2. Open in IntelliJ IDEA
3. Add JavaFX SDK via (File → Project Structure → Libraries)
4. Add VM options in (Run → Edit Configurations)
